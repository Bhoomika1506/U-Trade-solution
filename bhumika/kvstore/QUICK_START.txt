═══════════════════════════════════════════════════════════════════════════════
 🎯 QUICK EXAMPLES
═══════════════════════════════════════════════════════════════════════════════

Basic Operations:
  SET user:1 "Alice" EX 300    └─ Set with 5-min TTL
  GET user:1                   └─ Retrieve value
  TTL user:1                   └─ Check remaining time
  KEYS user:*                  └─ Pattern matching

Integers:
  INCR counter                 └─ Increment
  DECR counter                 └─ Decrement

Lists:
  LPUSH mylist a b c           └─ Push to head
  RPOP mylist                  └─ Pop from tail
  LLEN mylist                  └─ List length

Persistence:
  SAVE backup.json             └─ Save snapshot
  LOAD backup.json             └─ Restore snapshot
